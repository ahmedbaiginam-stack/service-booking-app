# 🏠 Service Booking Application

A full-stack **Service Booking System** built using Spring Boot and MongoDB where users can browse companies, view services, and book jobs.

---

## 🚀 Tech Stack

**Backend:**
- Java
- Spring Boot
- Spring Data MongoDB
- REST APIs

**Database:**
- MongoDB
